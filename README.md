## 🏗️ Currently Building
 
### MechanicShop — Workshop Management System
A full-stack system for managing a car repair workshop — customers, repair tasks, scheduling, invoicing, and a real-time dashboard.
 
**What's under the hood:**
- Clean Architecture with strict layer separation
- CQRS via MediatR with pipeline behaviours (validation, logging)
- EF Core + Repository + Unit of Work patterns
- JWT authentication + role-based authorization
- Blazor WASM frontend
- Dockerized services orchestrated with Docker Compose
- CI pipeline via GitHub Actions
> Currently in active development. Repository will be published upon completion.
